# Algoritmo de Codificação de Huffman

Implementação do Algoritmo de Huffman para compressão de dados sem perdas. Este projeto foi desenvolvido para a disciplina de Projeto e Análise de Algoritmos do curso de Engenharia de Computação da FAINOR.

## Descrição

O software executa o processo completo de compressão de texto baseando-se na frequência de ocorrência dos caracteres. O algoritmo garante que os caracteres mais frequentes recebam códigos binários menores, otimizando o armazenamento.

## Funcionalidades

* Análise de frequência de caracteres no texto de entrada.
* Construção da Árvore de Huffman (utilizando fila de prioridade).
* Geração da tabela de códigos binários.
* Codificação (compressão) e Decodificação do texto.

## Resultados

Nos testes realizados, o algoritmo obteve taxas de compressão entre 73% e 81%, dependendo da repetição de caracteres na entrada.
